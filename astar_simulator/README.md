# A* Simulator

## Dependencies
* Qt5
* Eigen

## Compiling
```bash
$ git clone https://github.com/vidit98/Astar-simulator
$ cd astar_simulator/
$ mkdir build && cd build
$ cmake ..
$ make
```
Run the exectuable as
```
$ ./rrt-test
```
