# RRT Simulator

## Dependencies
* Qt5
* Eigen

## Compiling
```bash
$ git clone https://github.com/sourishg/rrt-umass
$ cd rrt-umass/
$ mkdir build && cd build
$ cmake ..
$ make
```
Run the exectuable as
```
$ ./rrt-test
```
